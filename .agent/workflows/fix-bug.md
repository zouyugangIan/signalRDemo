# Bug Fixing Workflow

Follow these steps to ensure a robust fix:

1. **Reproduction**: Create a minimal reproduction case (test or script).
2. **Analysis**: Use the **Project Oracle** skill to trace the root cause.
3. **Plan**: Write an implementation plan artifact.
4. **Fix**: Apply the code changes.
5. **Verify**: Use the **Code Guardian** skill to run tests and verify the fix.
6. **No Regression**: Run the full test suite to ensure no other parts are broken.
